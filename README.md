# Discord Oauth2 Example

Dependencies:
  * express
  * url
  * axios
  * path
  * body-parser
  * node-fetch

To install these dependencies, run the following in your terminal (Be sure you have node.js and NPM installed.):
* `npm i express`
* `npm i url`
* `npm i axios`
* `npm i path`
* `npm i body-parser`
* `npm i node-fetch`
